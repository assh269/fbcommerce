🛍️ fbtiktokcommerce Workflow

How the platform connects Facebook/TikTok sellers with buyers via Telegram + Web

📌 Phase 1 — Seller Onboarding

1 Seller opens Telegram Bot → sends /start Bot
2 Taps "Register as Seller" → enters business name, phone, description Bot
3 Seller logs into Web Dashboard → adds products (name, price, stock, category) Web
4 Seller shares their Telegram bot link on Facebook & TikTok posts/livestreams Facebook/TikTok
⬇️ Buyer discovers products ⬇️

📌 Phase 2 — Buyer Orders via Telegram

1 Buyer sees product on Facebook post or TikTok livestream FB/TikTok
2 Buyer taps seller's Telegram bot link → opens bot Bot
3 Buyer browses catalog → taps product → "Add to Cart 🛒" Bot
4 Buyer opens cart → taps "Checkout ✅" Bot
5 Bot asks: Name → Phone → Address → Notes → Order placed! Bot
⬇️ Seller fulfills ⬇️

📌 Phase 3 — Seller Fulfills Order

1 Seller opens Web Dashboard → Orders — sees new order with buyer details Web
2 Seller clicks "Mark as confirmed" → calls buyer to confirm Web
3 Seller ships item → clicks "Mark as shipped" Web
4 Buyer receives → Seller clicks "Mark as delivered" Web
⬇️ Buyer reviews ⬇️

📌 Phase 4 — Trust & Reviews

1 Buyer can leave a ⭐ rating & review on the product (via bot or web) Bot Web
2 Reviews build seller reputation → more buyer trust → more sales Flywheel

💡 Key insight: Facebook & TikTok are used for discovery. Telegram bot handles the transaction. Web dashboard handles management.

Zero fees. No app download needed. Works on any phone.
